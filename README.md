# WireWareClient 🟠

A lightweight, fully vanilla Minecraft launcher built for performance — not bloat.

WireWareClient skips the stuff you don't need and gets you into the game fast. No giant version libraries, no heavy frameworks — just the most popular versions, ready to boot.

---

## Features

- **Vanilla-first** — no forced modding, no extra junk
- **Curated version list** — only the most popular versions included to keep the install size sane
- **Mod Modules** *(coming later)* — optional mod browser and installer planned for future releases
- **Offline identity support** — set a local alias for offline/LAN play
- **Radmin VPN integration** — displays your network info for easy private server setup
- **RAM & Java configuration** — allocate memory and set custom Java runtime paths
- **Clean UI** — custom-built dark interface, no asset packs

---

## Status

> ⚠️ Early release. Core launching works. Mod support is not yet implemented. Expect rough edges.

---

## Installation

1. Download the latest release from the [Releases](../../releases) page
2. Run the `.exe` — no installation needed
3. Set your alias on first launch and pick a version to boot

---

## Building from Source

Requirements:
- Visual Studio 2022
- .NET 8.0 SDK
- Windows 10/11

Clone the repo and open `WireWareClient.sln` in Visual Studio. Build and run.
